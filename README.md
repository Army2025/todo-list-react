# ✅ To-Do List

Aplicação de gerenciamento de tarefas desenvolvida com React e TypeScript.

## 🚀 Funcionalidades

- Adicionar tarefas
- Editar tarefas
- Excluir tarefas
- Marcar tarefas como concluídas
- Filtrar por Todas, Pendentes e Concluídas
- Persistência dos dados com LocalStorage

## 🛠️ Tecnologias utilizadas

- React
- TypeScript
- Vite
- CSS
- Git e GitHub
- Vercel
- LocalStorage
